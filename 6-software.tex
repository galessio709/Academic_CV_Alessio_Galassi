\section{Software tool}
    
    \begin{twocolentry}{\href{https://pros.unicam.it/theoc/}{pros.unicam.it/theoc}}
        \textbf{The O.C.$^3$} \mbox{(The Object-Centric Compliance Checking Tool)} 
    \end{twocolentry}  

    \begin{onecolentry}
        It is a web application for the definition of object-centric compliance rules (OCCR) and the application of such rules against an object-centric event log.
        The graphical interface enables users to: load an object-centric event log in JSON format, create and apply OCCR rules to generate matching and non-matching event sets.
    \end{onecolentry}
    
    \abitspace

    \begin{twocolentry}{\href{https://pros.unicam.it/coblock/}{pros.unicam.it/coblock}}
        \textbf{CoBlock} 
    \end{twocolentry}  
    
    \begin{onecolentry}
        It is a Domain-Specific Language (DSL) for defining compliance rules in terms of blockchain characterizations, treated as first-class citizens. The main advantage of the proposed DSL is that it allows auditors to focus on blockchain characterizations relevant to decentralized applications auditing, using compliance checking, and addressing the expressiveness limitations of existing compliance rules languages.
    \end{onecolentry}
